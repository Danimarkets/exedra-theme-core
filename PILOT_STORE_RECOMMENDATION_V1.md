# Exedra Theme Core — Pilot Store Recommendation V1

## Objective
Choose the best first real implementation target to validate Exedra Theme Core as a mature agency product.

## Stores considered
- miiscosmetics.com
- mytechcell.com.co
- carinosanto.com
- disfragancias.com

## Recommended pilot
# Miis Cosmetics

## Why Miis is the best pilot first
### 1. Best vertical fit with current strengths
Exedra Theme Core is currently strongest in:
- beauty / visual DTC
- editorial storytelling
- before/after potential
- merchandising blocks
- premium content layout
- PDP-led conversion structure

Miis matches that extremely well.

### 2. High richness of use cases
Miis can validate multiple important families at once:
- home editorial flow
- featured collections
- featured products
- image comparison potential
- product tabs
- testimonials / FAQ
- sticky ATC
- merchandising structure

### 3. Strong pressure test for real complexity
Miis also exposes real implementation pressure that is useful:
- catalog depth
- strong visual identity needs
- promotional logic
- B2B adjacency in the wider business context
- need for premium perception without losing commercial usability

### 4. Better than the other options as first pilot
#### MyTech
Strong candidate later, but better for second validation.
Why not first:
- current core is still more naturally beauty/fashion premium than spec-heavy tech-first
- the theme can support tech, but beauty is the cleaner first fit

#### Cariño Santo
Also strong, especially for fashion.
Why not first:
- fashion can demand stronger lookbook choreography and collection styling nuance
- better as second or third validation after the beauty pilot hardens the system

#### Disfragancias
Could work, but Miis likely offers richer product storytelling and stronger premium DTC validation value first

## Pilot success criteria
The pilot should validate:
- implementation speed
- editor usability
- section flexibility without custom rework explosion
- strong beauty presentation quality
- real PDP usability
- visual premium perception
- maintainability after first round of changes

## Expected output of pilot
After Miis pilot, Exedra Theme Core should move toward:
- Stable Release Candidate
- Better beauty preset layer
- Cleaner optional/premium boundaries
- Harder rules for which shells remain non-default
