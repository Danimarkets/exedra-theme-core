# Exedra Theme Core — Starter Pack V1

## Objective
Define the safest and most effective subset of the theme for a first real client implementation.

## Recommended default stack
### Global
- announcement-bar
- header
- footer

### Home
- hero
- image-with-text
- featured-collections-editorial
- featured-collection
- featured-products
- testimonials
- faq

### Collection
- main-collection-product-grid

### Product
- product-main
- product-tabs

### Conversion
- cart-drawer
- sticky-atc (optional)
- marquee (optional)

## Best-fit verticals for Starter Pack V1
- Beauty
- Fashion
- Visual DTC ecommerce

## Why this starter pack
- Strong enough visually
- Clear conversion structure
- Good compatibility with current app stack
- Lower risk than leading with shell modules
- Enough modularity for pilot deployment without overcomplication

## Modules to avoid in the first pilot unless specifically needed
- search-drawer
- promo-popup
- pickup-availability
- lookbook-slider
- product-bundles-shell
- related-products
