## Default block guidance
### 0 = Empty by default
Use for shells, integrations, and logic-heavy sections.

### 1 = Minimal demo
Use 1 clear example block only.
Best for:
- premium/optional sections
- media-heavy sections
- sections where too much demo would feel noisy

### 2 = Guided demo
Use 2–4 blocks depending on layout.
Best for:
- core editorial sections
- trust sections
- merchandising sections
- FAQ / testimonials / showcases

### 3 = Rich demo
Reserve only for curated showcase presets later.
Do not use widely in the neutral core.

## Recommended block counts
- Hero: 1 structured state
- Trust bar: 3 items
- Testimonials: 3 items
- FAQ: 3 items
- Multicolumn: 3 items
- Image gallery: 4 images
- Featured products: 4 products
- Featured collection: 4 placeholders max
- Collection showcase: 3 cards
- Products showcase: 4 products
- Lookbook cards: 3 cards

## Rule
If a section is not self-explanatory with demo content, increase clarity.
If a section becomes visually noisy with demo content, reduce intensity.
